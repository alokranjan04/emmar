# EMMAR Properties Landing Page with Voice AI Bot

This project contains a premium landing page for EMMAR Properties with an integrated Voice AI Bot.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
