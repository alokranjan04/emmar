# EMMAR Properties - Luxury Real Estate Landing Page

A premium, high-performance landing page for EMMAR Properties, built with modern web technologies to deliver a "wow" experience. This site showcases luxury developments, lifestyle experiences, and investor value propositions through a sophisticated, responsive interface.

## 🌟 Core Features

- **Immersive Hero Section**: Features a parallax background and smooth typography animations.
- **Featured Developments**: Interactive showcase of flagship communities like Downtown Dubai and Dubai Hills Estate.
- **Lifestyle Experience**: Curated sections highlighting the premium amenities and living standards.
- **Investor Metrics**: Real-time counter animations demonstrating Emaar's global impact and ROI.
- **Global Presence**: Interactive visualization of international reach.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile to ultra-wide displays.
- **Motion & Animations**: Utilizes `motion` (formerly Framer Motion) for subtle, premium micro-interactions.

## 🛠️ Technical Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19+](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion/React](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- NPM or PNPM

### Installation

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   - Copy `.env.example` to `.env.local`
   - Add your `GEMINI_API_KEY` (if used for AI features)

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components (Hero, Navbar, Footer, etc.).
- `/lib`: Utility functions and shared logic.
- `/hooks`: Custom React hooks.
